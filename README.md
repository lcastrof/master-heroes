## ğŸ’» Getting Started
### Requirements
* [yarn](https://yarnpkg.com/)

### Install
```bash
# Clone repository
$ git clone https://github.com/lcastrof/master-heroes

# Access directory
$ cd master-heroes

# Install dependecies
$ yarn install
```

### Running
```bash
$ yarn start
```

### Configurations
You can change the total cards to be displayed by changing the variable ```total_heroes_cards``` in the config file.
